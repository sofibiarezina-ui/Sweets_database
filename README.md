# Candy & Christmas Gifts Manager

Консольное приложение на языке Си (C99) для учета кондитерских изделий (сладостей) и их категорий, а также для **автоматического распределения всей базы сладостей по независимым новогодним подаркам** на основе введенных критериев.

## Основные возможности

- **Управление БД:** Полный CRUD (добавление с авторасширением памяти `realloc`, редактирование, «мягкое» логическое удаление без разрушения индексов).
- **Связывание данных:** Автоматическая линковка сладостей к их категориям через массивы указателей.
- **Динамическая сортировка:** Быстрый просмотр данных по любому критерию (цена, вес, сахар, количество элементов) без копирования структур.
- **Многокритериальный поиск:** Фильтрация сладостей по пороговой стоимости и содержанию сахара.
- **Интеллектуальное расформирование (Пункт 8):** Распределяет **абсолютно все** сладости из базы в массив списков подарков так, чтобы ни один подарок не превысил лимиты по весу, цене и числу уникальных типов.
- **Сортировка подарков:** Автоматически упорядочивает сформированные подарки **по возрастанию суммарного содержания сахара (в кг)**.
- **Импорт/Экспорт:** Чтение из бинарных файлов (`.bin`) и выгрузка отчетов по подаркам в текстовый файл (`.txt`).

## Структура проекта

| Файл | Описание |
|------|----------|
| `main.c` | Точка входа, диспетчер главного меню и состояние программы |
| `functions.h` | Структуры данных и прототипы |
| `display.c` / `tables_print.c` | Отрисовка консольных меню и выравнивание таблиц |
| `file_io.c` | Чтение бинарных файлов и связывание данных |
| `sort.c` / `sorted_lists_update.c` | Компараторы и управление индексами сортировки |
| `operations.c` | Бизнес-логика: добавление, фильтрация, алгоритм генерации подарков и экспорт отчетов |
| `free.c` | Каскадная очистка всей динамической памяти |

## Сборка и запуск

Проект написан на стандартном Си и не требует внешних библиотек. Для корректного отображения кириллицы в Windows используется кодировка UTF-8 (`SetConsoleOutputCP(65001)`).

**Компиляция через GCC (Linux/MinGW):**

```bash
gcc -std=c99 *.c -o candy_manager
./candy_manager
